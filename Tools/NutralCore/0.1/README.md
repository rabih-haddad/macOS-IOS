# NutralCore v0.1

**Release date:** February 18, 2026  
**Compatible with:** macOS 12.7.6 Monterey and later.

NutralCore is a modern macOS screensaver that visualizes AI training simulations through dynamic, HTML-based neural network animations. This initial release brings a clean, customizable experience.

## What's New

- First public release!
- Four built-in simulator styles (Simulator 1–4) selectable via the Options panel
- Persistent user selection using UserDefaults (remembers your choice)
- Modern WKWebView rendering with JavaScript enabled
- Simple, elegant configuration sheet with immediate preview reload

## Installation

1. Download [NutralCore-0.1.saver.zip](https://raw.githubusercontent.com/rabih-haddad/macOS-IOS/main/Tools/NutralCore/0.1/NutralCore-0.1.saver.zip)
2. Unzip → double-click `NutralCore.saver`
3. Click **Install** in the dialog
4. Open **System Settings → Wallpaper → Screen Saver**
5. Select **NutralCore**
6. Click **Options…** to choose your preferred simulator
